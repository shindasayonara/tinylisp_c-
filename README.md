# TinyLisp C++ 

## О проекте

Объектно-ориентированная версия интерпретатора Lisp на C++. Переписан с C на C++ для устранения проблем с глобальными переменными. Поддерживает создание нескольких независимых экземпляров интерпретатора.

## Особенности

- Полностью инкапсулированное состояние
- Можно создавать несколько интерпретаторов одновременно  
- Совместим с синтаксисом оригинального tinylisp
- Использует NaN-боксирование для эффективного хранения данных

## Структура файлов

- tinylisp.hpp # Заголовочный файл класса
- tinylisp.cpp # Реализация класса
- main.cpp # Пример использования

## Пример использования

```bash
#include "tinylisp.hpp"

int main() {
    TinyLisp lisp;
    
    // Вычисление выражений
    std::cout << lisp.eval("(+ 2 3)") << std::endl;  // 5
    std::cout << lisp.eval("(* 4 5)") << std::endl;  // 20
    
    // Работа со списками
    std::cout << lisp.eval("(cons 1 nil)") << std::endl;
    
    return 0;
}
```
